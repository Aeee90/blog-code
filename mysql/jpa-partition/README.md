# JPA (Hibernate) 환경에서의 파티셔닝 (feat. MySQL)

데이터 특성상 주기적인 삭제 작업이 필요한 경우 등이 파티션이 필요한 대표적인 예



파티셔닝은 DB에 의존적인 기술이다.  
즉, Hibernate에서 별도로 특별한 작업을 해야할 게 없다.  

테이블 생성시에만 파티셔닝을 위한 별도의 쿼리가 필요하며
이후 CRUD에 한해서는 별도로 처리할 필요가 없음
